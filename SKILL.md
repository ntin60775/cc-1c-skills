# 1c-devtools

Агрегатор навыков для полного цикла разработки на платформе 1С:Предприятие 8.3.

## Состав

Каждый поднавык в `.agents/skills/` покрывает узкую операцию:

- **cfe-*** — расширения конфигурации (CFE)
- **cf-*** — конфигурации
- **db-*** — информационные базы
- **epf-*** / **erf-*** — внешние обработки и отчёты
- **form-*** — управляемые формы
- **meta-*** — метаданные
- **subsystem-*** — подсистемы
- **template-*** / **mxl-*** — макеты
- **role-*** — роли
- **skd-*** — система компоновки данных
- **interface-*** — интерфейс
- **web-*** — веб-публикация и тестирование

## Использование

После установки навыки доступны через `$<имя-навыка>` (например, `$cfe-init`).
